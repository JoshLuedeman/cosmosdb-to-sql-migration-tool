ForWebActivity(string, int, int, bool, bool)

Emits the ADF policy block for a Web (failure-notification) activity. Defaults to secureInput and secureOutput so the webhook URL never enters ADF run history.

public static Dictionary<string, object?> ForWebActivity(string timeout = "00:05:00", int retry = 0, int retryIntervalInSeconds = 30, bool secureInput = true, bool secureOutput = true)

Parameters

timeout string

Activity timeout in HH:MM:SS format. Default "00:05:00".

retry int

Retry count. Default 0 — if the webhook is down, re-throwing is more useful than queuing retries.

retryIntervalInSeconds int

Seconds between retries. Default 30.

secureInput bool

Redact inputs from run history. Default true.

secureOutput bool

Redact outputs from run history. Default true.

Returns

Dictionary<string, object>

A serializer-ready dictionary representing the ADF policy JSON object.
