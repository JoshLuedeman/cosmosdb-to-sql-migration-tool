Remarks

Consent gates writing new outcomes and any telemetry transmission. Reading previously stored local outcomes (GetOutcomesAsync(CancellationToken)) is not gated: it is the user's own, already-stored, local data used to refine the user's own recommendations. To purge it, delete the file at StoreLocation.
