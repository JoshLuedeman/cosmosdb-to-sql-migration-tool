Thread-safe shared blackboard through which assessment agents communicate. Agents read the outputs produced by upstream agents, contribute their own domain output exactly once, append log messages, and record their run results. Implemented by SharedAssessmentContext.

public interface ISharedAssessmentContext

Remarks

The contract is the stable surface later agents and orchestration layers depend on. All members are safe to call concurrently. Collection-returning members return point-in-time snapshots, never live views.

ToAssessmentResult()

Projects the accumulated outputs into the existing AssessmentResult model so the unchanged downstream report / SQL-project / Data Factory generation can consume them. Best-effort: missing required outputs are filled with empty defaults rather than throwing, mirroring the single-pass pipeline's tolerance of a skipped data-quality phase.
