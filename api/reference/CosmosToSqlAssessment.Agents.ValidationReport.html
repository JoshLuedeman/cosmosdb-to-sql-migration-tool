Remarks

This is run metadata, not a domain assessment output: it is not listed by GetMissingRequiredOutputs() and it does not feed ToAssessmentResult(). A missing report (HasValidationReport is false) means validation never ran or crashed before it could emit a report — consumers should treat that as an infrastructure failure distinct from an unacceptable-but-reported run.

Consumers should prefer the typed IsAcceptable/IsComplete verdict over parsing Messages.
