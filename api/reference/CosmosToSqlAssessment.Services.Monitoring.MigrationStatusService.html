Remarks

Pulls progress samples from an IMigrationStatusSource (backed by ADF run telemetry from parents #70/#76), enriches them through the same MigrationMonitoringService derivation used by the publishing path (#223), and writes a concise per-update line plus a closing summary to a caller-supplied TextWriter.

Status reporting is strictly read-only: it derives snapshots with a NullMigrationMetricPublisher so viewing progress never re-publishes custom metrics to Azure Monitor.
