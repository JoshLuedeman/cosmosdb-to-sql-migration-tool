A local, append-only, JSON-lines (.jsonl) implementation of IFeedbackStore. Each anonymized MigrationOutcome is serialized to a single line. This is the default, fully-offline storage mechanism for the feedback loop.

Remarks

Writes are serialized with a process-local semaphore and use an append-mode FileStream with Read; reads open the file with ReadWrite and stream line-by-line without buffering the whole file. Malformed or truncated lines (e.g., from an interrupted write) are skipped and logged without echoing their raw content.
