Generates ready-to-deploy Azure Data Factory artifacts (linked services, datasets, pipelines) from an AssessmentResult. Foundational sub-issue #141 of parent #70: copy activities per container → table mapping. Later sub-issues extend the output with parameters, retry policy, monitoring, validation, ARM templating, and incremental-load support.

Methods

GenerateAsync(AssessmentResult, string, DataFactoryGenerationOptions?, CancellationToken)

Generates all ADF linked services, datasets, and pipeline JSON files from assessment and writes them under outputDirectory.

Task<DataFactoryGenerationResult> GenerateAsync(AssessmentResult assessment, string outputDirectory, DataFactoryGenerationOptions? options = null, CancellationToken cancellationToken = default)

Parameters

assessment AssessmentResult

Assessment result containing the container-to-table mappings to translate into ADF Copy activities.

outputDirectory string

Root directory under which the ADF/ folder tree is created.

options DataFactoryGenerationOptions

Generation toggles; defaults to a foundational full-load configuration when null.

cancellationToken CancellationToken

Token to observe for cooperative cancellation.

Returns

Task<DataFactoryGenerationResult>

A DataFactoryGenerationResult listing every file written and any warnings the operator should review.

Examples

using Microsoft.Extensions.DependencyInjection;

IDataFactoryPipelineGenerator generator =
    serviceProvider.GetRequiredService<IDataFactoryPipelineGenerator>();

DataFactoryGenerationResult result =
    await generator.GenerateAsync(assessmentResult, outputDirectory: "out");

foreach (var file in result.GeneratedFiles)
    Console.WriteLine($"Wrote {file}");
