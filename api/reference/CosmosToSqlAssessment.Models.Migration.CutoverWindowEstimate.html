Estimated cutover downtime window for an online (change-feed-based) migration (#136). Models the final maintenance window during which the source is quiesced (read-only): quiesce overhead, final delta-sync drain of the residual change-feed backlog, data validation, connection switchover, plus a safety buffer. Consumes the #135 IncrementalSyncEstimate. Grounds the phased plan (#137) and report runbook (#139).
