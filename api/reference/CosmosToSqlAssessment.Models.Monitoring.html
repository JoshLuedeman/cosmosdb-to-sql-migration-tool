Namespace CosmosToSqlAssessment.Models.Monitoring

Classes

AlertRuleOptions

Strongly-typed options controlling the Azure Monitor alert-rule ARM templates emitted for the migration custom metrics (#223) and the orchestrating Data Factory pipelines (#70). Bound from the AzureMonitor:Alerts configuration section.

AnomalyDetectionOptions

Options for the rolling-window z-score anomaly detector (#226) applied to migration Request-Units/sec and throughput. Bound from the AzureMonitor:Anomaly section.

MigrationAnomaly

A single detected anomaly in a migration metric (Request-Units/sec or throughput) relative to a rolling-window statistical baseline. Produced by the anomaly detection service (#226) and surfaced as a warning by the migration status command (#225).

MigrationMetricPoint

A single custom-metric data point destined for Azure Monitor. The AzureMonitorMetricPayloadBuilder groups points that share a namespace, name, timestamp, and dimension shape into one ingestion payload.

MigrationProgressSample

A single point-in-time observation of an in-flight migration, typically derived from an Azure Data Factory pipeline/activity run (parent #70). Each sample carries the delta work completed since the previous sample for the same pipeline/activity/run so the monitoring pipeline can derive both windowed rates and cumulative totals.

MigrationProgressSnapshot

An enriched, consumer-facing view of a single MigrationProgressSample. Carries the windowed values from the sample plus cumulative totals and derived rates for the originating pipeline/activity/run, and the custom-metric points that were emitted for the sample. Produced by the migration monitoring pipeline and consumed by the migration status CLI command (#225) and anomaly detection (#226).

MigrationStatusReportOptions

Options controlling the migration status CLI command (#225): whether to keep watching and how frequently to poll.
