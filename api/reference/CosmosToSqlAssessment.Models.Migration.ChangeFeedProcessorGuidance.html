Database-level Change Feed Processor (CFP) implementation guidance (#140 of parent #69). Produced from the #134 change-feed availability analysis, it advises on building a CFP worker as an alternative (or complement) to the Azure Data Factory _ts-watermark incremental copy pipeline this tool also generates. All figures are conservative planning guidance, not validated sizing — monitor and adjust.
