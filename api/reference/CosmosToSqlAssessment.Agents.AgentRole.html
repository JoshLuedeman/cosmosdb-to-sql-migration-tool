Identifies the assessment domain an IAssessmentAgent owns and the kind of output it contributes to a SharedAssessmentContext.

public enum AgentRole

Fields

CosmosAnalysis = 0

Analysis of the source Cosmos DB account, databases, and containers.

SqlPlanning = 1

Azure SQL platform recommendation and migration planning.

DataQuality = 2

Pre-migration data-quality analysis of the source documents.

DataFactoryEstimation = 3

Azure Data Factory migration estimate derived from the Cosmos analysis and SQL plan. Produced by the orchestrator as a derived step rather than by a domain agent.

Validation = 4

Cross-checking and completeness validation of other agents' outputs.

Orchestration = 5

Coordination of the agent run itself (used by the orchestrator).

Remarks

Roles let the orchestrator and the validator reason about which domain produced a given result without depending on concrete agent types. Adding new values is a source-compatible change, so the enum can grow as later epics introduce more agents.
