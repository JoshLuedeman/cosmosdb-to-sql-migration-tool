CosmosDB to SQL Migration Tool
Table of Contents

Namespace CosmosToSqlAssessment.Services.Migration

Classes

ChangeFeedAvailabilityAnalyzer

Determines per-container change feed readiness for incremental (change-feed-based) migration. Pure analysis over an already-collected CosmosDbAnalysis; performs no live Cosmos DB calls so it is deterministic, fast, and unit-testable. Foundational sub-issue #134 of parent #69 (incremental migration support with change feed).

ChangeFeedProcessorGuidanceGenerator

Produces Change Feed Processor (CFP) implementation guidance (#140 of parent #69) from the already computed #134 change-feed availability analysis. Pure composition; performs no live Cosmos DB calls.

CutoverWindowCalculator

Calculates the estimated cutover downtime window for an online (change-feed-based) migration (#136 of parent #69). Pure analysis over the #135 IncrementalSyncEstimate; performs no live Cosmos DB calls.

IncrementalSyncEstimator

Estimates and compares the initial bulk-load duration against change-feed incremental sync time and steady-state behavior (#135 of parent #69). Pure analysis over the already-collected CosmosDbAnalysis and DataFactoryEstimate; performs no live Cosmos DB calls and reuses the existing Data Factory initial-load estimate as its baseline.

PhasedMigrationPlanGenerator

Synthesizes a phased incremental (change-feed-based) Cosmos DB → SQL migration plan (#137 of parent #69) from the already-computed change-feed availability (#134), incremental sync estimate (#135), and cutover window (#136). Pure composition; performs no live Cosmos DB calls.

TimeBasedPartitioningAnalyzer

Recommends Azure SQL time-based table partitioning strategies and Cosmos _ts-based initial-load slicing for each migrated container (#138 of parent #69). Pure analysis over the already-collected CosmosDbAnalysis; performs no live Cosmos DB calls.
