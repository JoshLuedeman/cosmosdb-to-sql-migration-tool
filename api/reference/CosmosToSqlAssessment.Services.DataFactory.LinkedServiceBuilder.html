BuildAzureSqlLinkedService(AdfNameRegistry, DataFactoryGenerationOptions)

Builds the shared Azure SQL linked service used by every generated sink dataset. Authentication is controlled by UseManagedIdentityForSql and UseAzureKeyVault: MI (default) emits server + database + authenticationType=SystemAssignedManagedIdentity; Key Vault emits a connection string with an AKV password reference; otherwise a placeholder connection string is written.

BuildCosmosLinkedService(string, AdfNameRegistry, DataFactoryGenerationOptions)

Builds a parameterised Cosmos DB linked service for databaseName. Authentication shape is controlled by UseManagedIdentityForCosmos and UseAzureKeyVault: MI (default) emits accountEndpoint + database; Key Vault emits a connection string with an AKV secret reference; otherwise a placeholder connection string is written.
