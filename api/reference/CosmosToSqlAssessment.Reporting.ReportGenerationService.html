GenerateAssessmentReportAsync(AssessmentResult, string, CancellationToken)

Generates a comprehensive assessment report with separate Excel files per database and one Word summary Follows Azure documentation standards and best practices

public Task<(List<string> ExcelPaths, string WordPath, string AnalysisFolderPath)> GenerateAssessmentReportAsync(AssessmentResult assessmentResult, string outputDirectory, CancellationToken cancellationToken = default)

Parameters

assessmentResult AssessmentResult

The completed assessment to render. For multi-database runs, populate IndividualDatabaseResults to emit one Excel file per database.

outputDirectory string

Base directory for output; a timestamped analysis folder is created beneath it. When null, the Reporting:OutputDirectory configuration value (default "Reports") is used.

cancellationToken CancellationToken

Token to observe for cooperative cancellation.

Returns

Task<(List<string> ExcelPaths, string WordPath, string AnalysisFolderPath)>

A tuple of the generated Excel file paths, the Word summary path, and the analysis folder that contains them.

Examples

using Microsoft.Extensions.DependencyInjection;

var reports = serviceProvider.GetRequiredService<ReportGenerationService>();

// The result tuple deconstructs into the generated artifact paths.
(List<string> excelPaths, string wordPath, string analysisFolder) =
    await reports.GenerateAssessmentReportAsync(assessmentResult, outputDirectory: "out");

Console.WriteLine($"Word summary:  {wordPath}");
Console.WriteLine($"Excel reports: {string.Join(", ", excelPaths)}");
Console.WriteLine($"All output in: {analysisFolder}");
