FeedbackCollectionService

Coordinates the opt-in collection of anonymized migration outcomes. Collection and any remote transmission are gated behind an explicit consent decision that defaults to disabled.

FeedbackConsent

The resolved feedback consent decision and the source that determined it.

FeedbackOptions

Strongly-typed configuration for the continuous-learning feedback loop, bound from the FeedbackLoop configuration section. Feedback is off by default.

RecommendationRefinementService

Correlates the current assessment with prior, anonymized MigrationOutcomes to refine (or confirm) the recommended Azure SQL platform and tier, producing an attributable RecommendationRefinement.

RefinementAbComparison

The aggregate result of an A/B comparison between the static baseline recommendation and the learning-refined recommendation across a labeled set of RefinementScenarios. Correctness is measured on the full (platform, tier) pair.

RefinementScenario

A single labeled scenario used to evaluate refinement accuracy: an assessment, the prior migration history available at the time, and the ground-truth configuration that is known to perform satisfactorily for that workload.

SqlMigrationAssessmentService

Service for assessing Cosmos DB to SQL migration requirements and recommendations Implements Azure best practices for SQL platform selection

SqlProjectGenerationService

Service for generating SQL Server Database Projects (.sqlproj) from migration assessments Creates SSDT-compatible projects for cloud-native Azure SQL Database deployment

SqlProjectIntegrationService

Service for integrating SQL Database Project generation with the main application workflow Provides orchestration between migration assessment and SQL project creation

ValidationScriptGenerationResult

Result of generating post-migration validation artifacts.

ValidationScriptGeneratorService

Generates post-migration validation artifacts (SQL + PowerShell + report templates) into the Scripts/PostMigration/ folder of a generated SQL Database Project.

WorkloadSimilarity

Computes a bounded similarity score between two anonymized WorkloadProfile fingerprints. The score weights normalized workload complexity and size most heavily, with smaller contributions from container count and provisioned throughput.
