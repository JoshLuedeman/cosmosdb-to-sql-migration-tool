Class DatasetBuilder

Namespace
CosmosToSqlAssessment.Services.DataFactory
Assembly
CosmosToSqlAssessment.dll

Builds ADF source / sink datasets for every Cosmos container and every target SQL table. Datasets declare parameters (e.g. databaseName, collectionName) and forward them into the linked-service reference and typeProperties so the same dataset can be reused across environments by passing parameter values from the invoking pipeline (#142).
