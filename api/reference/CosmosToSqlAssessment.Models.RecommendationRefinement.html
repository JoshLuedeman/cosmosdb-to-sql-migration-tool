The result of correlating the current assessment with prior, anonymized migration outcomes to refine (or confirm) the recommended Azure SQL platform and tier. Carries an attributable rationale ("based on N prior similar migrations") for transparent inclusion in reports.

Remarks

When HasRefinement is false the refined values equal the baseline and ChangedFromBaseline is false; the Rationale explains why no learning was applied (e.g., too little comparable history). When HasRefinement is true the recommendation may still equal the baseline — in which case the prior outcomes confirm rather than change it.
