A single point-in-time observation of an in-flight migration, typically derived from an Azure Data Factory pipeline/activity run (parent #70). Each sample carries the delta work completed since the previous sample for the same pipeline/activity/run so the monitoring pipeline can derive both windowed rates and cumulative totals.
