Remarks

Consumption contract: AssessmentResult is a best-effort projection that is always non-null but is only safe to drive downstream report / SQL-project / Data Factory generation when IsAcceptable is true. When IsAcceptable is false (a required output is missing or outputs are inconsistent), the result may contain empty defaults and consumers should surface the failure instead of generating artifacts.
