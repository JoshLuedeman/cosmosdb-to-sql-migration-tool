Immutable record of how a single agent run finished. Recorded on the SharedAssessmentContext so the orchestrator can isolate failures and the ValidatorAgent can reason about completeness.

Remarks

Contract: a Succeeded result guarantees the agent's required outputs were committed to the context. Failed and Skipped make no such guarantee — consumers must treat the corresponding outputs as possibly absent.
