Namespace CosmosToSqlAssessment.Services.Feedback

Classes

CoarsenedOutcome

A telemetry-safe, coarsened projection of a MigrationOutcome intended for the optional remote telemetry endpoint. Exact aggregate counts and costs are replaced with bucketed labels to further reduce any re-identification risk when data leaves the local machine.

HttpFeedbackTelemetrySink

An IFeedbackTelemetrySink that POSTs a coarsened payload as JSON to the configured TelemetryEndpoint. Transport failures are logged and swallowed so that local feedback collection always succeeds regardless of network conditions.

LocalJsonFeedbackStore

A local, append-only, JSON-lines (.jsonl) implementation of IFeedbackStore. Each anonymized MigrationOutcome is serialized to a single line. This is the default, fully-offline storage mechanism for the feedback loop.

NullFeedbackTelemetrySink

A no-op IFeedbackTelemetrySink used when no remote telemetry endpoint is configured. Nothing leaves the local machine.

Interfaces

IFeedbackStore

Abstraction over the durable store for anonymized MigrationOutcome records that power the continuous-learning feedback loop. Implementations must never persist any data beyond what is present on the supplied (already anonymized) outcome.

IFeedbackTelemetrySink

Optional transport that forwards a CoarsenedOutcome to a configured remote telemetry endpoint. Only invoked when feedback consent is granted and a telemetry endpoint is configured.
