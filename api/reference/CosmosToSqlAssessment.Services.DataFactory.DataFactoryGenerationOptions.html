IncrementalCopy

Incremental load configuration (#147). Default Enabled is false — the generator emits the same full-load artifacts as #146. When opted in, the orchestrator wraps each per-mapping Copy in a Lookup → SetVariable → Copy(query-override) → Script chain that maintains a _ts watermark per source-container → target-table mapping. The IncrementalCopyMode enum reserves a future ChangeFeedDataFlow slot for parent #69 to layer on a native change-feed Mapping Data Flow without breaking the option surface.
