Builds Azure Monitor custom-metric ingestion payloads from MigrationMetricPoints. The output matches the documented schema:

{ "time": "<ISO-8601 UTC>",
  "data": { "baseData": { "metric": "...", "namespace": "...",
    "dimNames": ["..."], "series": [ { "dimValues": ["..."], "min": n, "max": n, "sum": n, "count": n } ] } } }

Points that share a namespace, metric name, timestamp, and dimension-name shape are grouped into a single payload with one series per point. This keeps each payload valid (Azure Monitor requires a stable dimNames array across all series in a payload). The class is pure and deterministic, so it is exercised directly in unit tests without a live call.
