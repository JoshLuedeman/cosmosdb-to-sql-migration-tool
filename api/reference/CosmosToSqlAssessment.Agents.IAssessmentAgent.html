A single-responsibility participant in an agentic assessment run. Each agent owns one AgentRole, reads any upstream outputs it needs from the shared context, performs its work, and commits its own output back to the context.

public interface IAssessmentAgent

Remarks

This is the stable abstraction the orchestrator (#215) and later epics (#132/#133/#69) compose against. Implementations must:

  • be safe to construct via dependency injection and to run independently in isolation;
  • never throw for a recoverable condition — return a Failed(string, AgentRole, string, TimeSpan) or Skipped(string, AgentRole, string) result and log a message instead, so a single agent's problem does not abort the whole run;
  • commit their domain output exactly once via the context's Set* methods on success.
