Remarks

Privacy contract. This schema is anonymized and aggregate by construction: it contains no names, endpoints, identifiers (other than a random, non-correlatable OutcomeId), IP addresses, credentials, or free-text notes. Every field is either a closed enum, an aggregate count/size, a tool-generated categorical label, or a numeric metric. There is intentionally no property in which a caller could place customer data such as a database name or a sample document.

The three metric families required by the feedback loop are represented explicitly: success (Status, ActualMigrationDays, DataCompletenessPercent), performance (DeployedPlatform, DeployedTier, utilization/latency, PerformanceSatisfactory), and cost actual vs estimate (the monthly and one-time cost pairs).
