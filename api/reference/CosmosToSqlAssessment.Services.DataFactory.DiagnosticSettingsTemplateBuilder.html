Class DiagnosticSettingsTemplateBuilder

Emits a stand-alone, deployable ARM template containing a Microsoft.Insights/diagnosticSettings resource scoped to the target Data Factory. Sets logAnalyticsDestinationType = "Dedicated" so logs land in the resource-specific ADFPipelineRun / ADFActivityRun tables (matching the bundled KQL queries) rather than the legacy AzureDiagnostics table.
