An enriched, consumer-facing view of a single MigrationProgressSample. Carries the windowed values from the sample plus cumulative totals and derived rates for the originating pipeline/activity/run, and the custom-metric points that were emitted for the sample. Produced by the migration monitoring pipeline and consumed by the migration status CLI command (#225) and anomaly detection (#226).
