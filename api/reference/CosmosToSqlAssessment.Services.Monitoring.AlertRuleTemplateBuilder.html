Remarks

Two templates are produced:

  • A Microsoft.Insights/metricAlerts template covering threshold breaches (static error-rate), error spikes (a dynamic-threshold criterion), low/no throughput, and an optional Request-Units ceiling, all over the MigrationMonitoringService custom metrics.
  • A Microsoft.Insights/scheduledQueryRules log-alert template that detects stalled pipelines from the ADFPipelineRun/ADFActivityRun Log Analytics tables — the reliable dead-pipeline detector, because metric alerts do not fire on missing data.

Each Build… method returns a fully-formed ARM template object ($schema, contentVersion, parameters, resources) intended to be serialized with AdfJsonSerializer, which preserves dictionary keys such as odata.type verbatim.
