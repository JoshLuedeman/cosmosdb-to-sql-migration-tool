Remarks

The templates are deploy-time artifacts: they describe Microsoft.Insights/metricAlerts rules over the MetricNamespace custom metrics plus a Microsoft.Insights/scheduledQueryRules log alert over the ADFPipelineRun/ADFActivityRun Log Analytics tables. None of these settings trigger a live Azure call — they only shape the generated JSON.

Threshold and severity defaults are conservative starting points an operator is expected to tune; every value is surfaced as an ARM template parameter default so the emitted templates remain deployable and overridable without regeneration.
